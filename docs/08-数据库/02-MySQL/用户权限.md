
## 1.创建用户
```sql
create user '账号'@'访问主机地址 localhost | ip地址 | %' identified by '密码'
```
## 2.赋予权限
```sql
grant 权限,权限,权限 on 数据库名.表名(* 任何库和表) to '账号'@'主机地址'
```
## 3.回收权限
```sql
revoke 权限,权限,权限 on 数据库名.表名 from '账号'@'主机地址'
```
## 4.删除用户
```sql
drop user '用户名'
```
## 5.查看用户和权限
```sql
select user,host from mysql.user;
show grants fro '账号'@'主机地址'
```
