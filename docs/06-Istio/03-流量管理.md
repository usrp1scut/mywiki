## Istio入流量方向

* Client -> IngressController -> Gateway -> VirtualService -> Service -> workload

## 1.创建网关

```yaml
apiVersion: networking.istio.io/v1alpha3
kind: Gateway
metadata:
  name: mygw
spec:
  selector:
    istio: ingressgateway # use istio default controller
  servers:
  - port:
      number: 80
      name: http
      protocol: HTTP
    hosts:
    - "aa.xiebo.pro"
```

## 2.VirtualService用法

### 2.1.基础用法

```yaml
apiVersion: networking.istio.io/v1alpha3
kind: VirtualService
metadata:
  name: myvs
spec:
  hosts:
  - "aa.xiebo.pro"
  gateways:
  - mygw
  http:
  - route:
    - destination:
        host: svc1
```
### 2.2.前缀匹配转发访问目录

```yaml
apiVersion: networking.istio.io/v1alpha3
kind: VirtualService
metadata:
  name: myvs
spec:
  hosts:
  - "aa.xiebo.pro"
  gateways:
  - mygw
  http:
  - name: aaa
    match:
      - uri:
          prefix: /demo1
    route:
    - destination:
        host: svc1
  - name: bbb
    match:
      - uri:
          prefix: /demo2
    route:
    - destination:
        host: svc2
```
